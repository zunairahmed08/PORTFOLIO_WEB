## Portfolio Website

## Overview
This is a *professional, responsive, modern portfolio website* built by *Zunair Ahmed* using *HTML, CSS, and JavaScript*.  
It showcases personal information, skills, projects, and contact details in a clean and interactive layout.



## Features

- Home Section: Introduction with name, title, and profile picture  
- About Section*: Brief description about Zunair Ahmed  
- Skills Section: List of programming & development skills  
- Projects Section: Showcases demo projects with descriptions  
- Contact Section: Email and social media links   
- Smooth Animations: Fade-in effects and hover interactions  
- Responsive Design: Works perfectly on desktop, tablet, and mobile devices  


## Technologies Used

- HTML5: For website structure and semantic markup  
- CSS3*: For styling, layout, animations, and responsive design  
- JavaScript: For dark mode toggle and scroll-based animations

## Instructions to Run

1. *Download* or *clone* this repository.  
2. Make sure all files (index.html, style.css, script.js) are in the same folder.  
3. Open **index.html** in any web browser (Chrome, Edge, Firefox).  
4. Explore sections and try the *dark mode toggle* button.
